Python MCP Server

@mcp.tool()
def calculate_bmi(height_cm: float, weight_kg: float) -> dict:
    bmi = weight_kg / (height_cm / 100) ** 2
    return {"bmi": round(bmi, 2), "category": "正常"}

Node.js MCP Server

server.tool(
  'calculate_bmi',
  '计算 BMI',
  { height_cm: z.number(), weight_kg: z.number() },
  async ({ height_cm, weight_kg }) => {
    const bmi = weight_kg / Math.pow(height_cm / 100, 2);
    return { content: [{ type: 'text', text: JSON.stringify({bmi}) }] };
  }
);

JSON-RPC 2.0 消息格式

tools/list
{"jsonrpc":"2.0","method":"tools/list","id":1}
tools/call
{"jsonrpc":"2.0","method":"tools/call","params":{"name":"calculate_bmi","arguments":{"height_cm":170,"weight_kg":65}},"id":2}

💡 面试要点

  • • MCP = Model Context Protocol，AI 调用外部工具的标准化协议
  • • MCP Client 发现工具 → 调用工具 → 返回结果给 LLM
  • • 传输层支持 stdio（本地）和 SSE（远程）
  • • 优势：标准化、热插拔、跨框架
  • • vs 代码写死的 @tool：MCP 可以动态发现和添加工具

4 个服务同时运行：Python MCP (8100) + Node.js MCP (8101) + Python Agent (8200) + Node.js Agent (8201)
